NIST Compliance Audit Tool

A cybersecurity GRC portfolio project using NIST control concepts to review evidence, identify compliance gaps, and translate findings into practical remediation actions.

Project Overview

This project demonstrates how a lightweight compliance audit process can be used to review cybersecurity control evidence, identify gaps, and communicate risk in a clear and organized way. The project was designed as a professional portfolio artifact to show skills in governance, risk, compliance, documentation, evidence review, and cybersecurity communication.

The focus of this project is not to perform a full technical audit. Instead, it shows how control requirements can be reviewed against available evidence and translated into business-friendly findings that support remediation planning and audit readiness.

Project Highlights

Business Problem

Organizations often struggle to organize control evidence, identify compliance gaps, and explain cybersecurity risks in a way that is clear to both technical and non-technical stakeholders.

What I Built

I created a lightweight NIST compliance audit project that reviews control evidence, organizes findings, documents gaps, and connects each gap to practical remediation actions.

Skills Demonstrated

This project demonstrates governance, risk assessment, documentation, control review, evidence evaluation, remediation planning, and cybersecurity communication skills.

What I Built

I built a practical compliance review artifact that organizes cybersecurity control evidence into a structured assessment format. The project includes a control review summary, evidence notes, gap identification, risk ratings, remediation recommendations, and a source log.

The purpose was to show how cybersecurity documentation can support decision-making, audit readiness, and continuous improvement. The final deliverables help explain what was reviewed, what gaps were identified, and what actions should be taken next.

Skills Demonstrated

  • Cybersecurity governance, risk, and compliance documentation
  • NIST control interpretation and evidence review
  • Gap assessment and remediation planning
  • Risk rating and prioritization
  • Audit-ready source tracking and documentation
  • Clear communication of cybersecurity findings
