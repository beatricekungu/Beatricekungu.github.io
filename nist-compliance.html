Florida Public-Sector NIST Compliance Review

A GRC portfolio project using public Florida cybersecurity audit sources and NIST-aligned control mapping.

Project Overview

This project uses publicly available Florida public-sector cybersecurity audit resources to simulate a NIST-aligned Governance, Risk, and Compliance review. I reviewed public cybersecurity audit themes, mapped them to control areas, assigned compliance status and risk level, and created remediation actions to support audit readiness.

The project demonstrates my ability to translate cybersecurity governance concepts into practical documentation, evidence review, risk analysis, remediation planning, and leadership-ready reporting.

Business Problem

Public-sector organizations must demonstrate that cybersecurity controls are documented, implemented, monitored, and reviewed. Public audit reports often identify gaps in access reviews, policy governance, incident response, monitoring evidence, and remediation tracking.

These gaps can make audit preparation slower and make it harder for leadership to understand which risks should be prioritized first. This project simulates how a GRC analyst can organize audit evidence, identify control gaps, and create practical remediation actions.

What I Built

I created a structured control review package using public Florida cybersecurity audit themes and NIST-aligned control areas. The project includes a README, audit summary, CSV control tracker, source log, and live website page.

  • Mapped public audit themes to NIST-aligned control areas.
  • Assigned compliance status based on evidence quality.
  • Assigned risk levels based on business and security impact.
  • Created remediation actions to support audit readiness.
  • Documented source evidence for transparency and repeatability.
